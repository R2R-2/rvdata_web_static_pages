Navigating The Cruise Catalog

The R2R website offers several ways to access its data stores. Choosing the "BROWSE VESSELS" button in the upper navb bar takes you to the cruise catalog. Selecting a vessel will take you to the vessel page which includes a paginated list of all cruises ordered by date. Selecting a cruise will take you to the cruise page.

If you know ahead of time which cruise you would like to access then then you may enter it into the the "Search cruise" input area in the upper left sidebar. Doing so well take you to that cruise's page directly bypassing the cruise catalog interface completely.

The cruise page lists all filesets that belong to that cruise. Each fileset generally maps to a device although a single device may correspond to multiple filesets if it generates multiple data types. To access the direct download for a fileset use its . To check the contents of the fileset use the button. This page will list all filenames along with their byte counts and md5 hash values.

If instead of the button you see then the Chief Scientist of the cruise has not yet authorized the data for publishing and the data will remain unavailable until it has been released.

If R2R has obtained a data release from the Chief Scientist then the "Request Distro Files" button will appear under "Additional Cruise Information". The distro request tool allows you to request any subset of files from the entire data package (distro). R2R will reach out with instructions on how to access the requested data.

Access The Search Page

The "SEARCH" button in the upper nav bar takes you to the Search page which offers an alternative method for finding data: filtering through filesets directly. The Search page page allows you to access filesets across multiple cruises by searching with a variety of parameters including (but not limited to) device type, vessel name, and operating institution. For a rundown of device options the "DATA TYPES & PRODUCTS" button will take you to an informational page with descriptions of all device types that R2R serves data from.

The "Search device" and "Search keyword" search areas on the left sidebar are a shortcut for accessing the Search page.

Accessing And Using The QA Dashboard

In addition to serving data R2R also performs QA (quality assurance) to assess the validty of data documenting this in QA reports. To access these QA reports click "QA Dashboard" on the left side bar. To access a report for a specific cruise and enter it into the "Cruise" field and hit "Go". If you want to access QA reports across multiple cruises instead then use the "Vessel" dropdown to specify the vessel.

Once you've located a QA report you can view the documents with the "Docs & Plots" button. The Quality Assessment Report document contains useful information on the quality assessment criteria which are visualized as colored lamps on the QA dashboard interface.
